# 🐦 Flappy Bird — C++ Text Edition

A terminal-based Flappy Bird game built in C++ to demonstrate core OOP concepts.

## 🧠 Concepts Demonstrated
- Abstract Classes & Pure Virtual Functions
- Inheritance & Inheritance Hierarchy
- Destructors
- Multiple Inheritance
- Function Overloading
- Polymorphism
- Pointers

## ▶️ How to Run
```bash
g++ flappy_bird.cpp -o flappy_bird
./flappy_bird
```

## 🎮 Controls
- `SPACE` — Flap
- Works on Linux, macOS, and Windows
